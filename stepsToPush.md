## Steps To Push Code To GitHub
- Create a folder for code
```
mkdir <nameOfFolder>
```
- Create local repository
```
git init
```
- Create code
```
touch readme.md
```
- Save Code!!!

- Track/Stage/watch all of the folders
```
git add .

```
or

```
git add -A

```
- Commit our changes to our local git
```
git commit -m "initial commit"

```
- Create a GitHub repo
